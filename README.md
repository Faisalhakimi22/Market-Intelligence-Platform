Full-stack market intelligence and analytics platform with AI-powered insights, data visualization, and real-time analysis.

## 🚀 Features

- 📊 **Data Analytics**: Advanced market data analysis and visualization
- 🤖 **AI Insights**: OpenAI integration for intelligent analysis
- 📈 **Charts**: Interactive charts with Chart.js and Recharts
- 🔐 **Authentication**: Passport.js local authentication
- 💾 **Database**: PostgreSQL with Drizzle ORM
- ⚡ **Real-time**: WebSocket support for live updates

## 🛠️ Tech Stack

**Frontend:**
- React 18, Wouter routing
- TailwindCSS, Radix UI, shadcn/ui
- Chart.js, Recharts
- TanStack Query

**Backend:**
- Express.js, TypeScript
- Drizzle ORM, PostgreSQL (Neon)
- Passport.js authentication
- WebSocket (ws)

**AI/ML:**
- OpenAI API
- ml-matrix, simple-statistics

## 📦 Installation

```bash
npm install
npm run dev
```
